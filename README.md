# Agnidhra Technologies LMS

A React-based learning management system for AI and Cybersecurity programs.

## Features

- **AI Program**: Full-Stack AI Professional course with project-based learning
- **Cyber Program**: Full-Stack Cyber Professional course covering defensive and offensive security
- Responsive design with theme-specific styling
- Interactive syllabus accordions
- Portfolio-focused curriculum

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

```
src/
```
